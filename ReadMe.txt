Notre projet a été réalisé sur IntelliJ.

Nous représentons le graphe sous forme de noeud correspondant aux mots et d'arêtes correspondant aux relations entre deux mots.

L'exécution du main donne les réponses aux questions 1 à 6 pour l'ensemble des graphs (mais c'est long (surtout sur mon pc)).

Nous avons réalisé une méthode de sérialisation afin de sauvegarder les graphes pour que le code s'exécute plus vite.
